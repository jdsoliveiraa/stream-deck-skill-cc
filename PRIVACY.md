# Privacy Policy

**stream-deck** Claude Code Plugin

Last updated: 2026-03-30

## Data Collection

This plugin does **not** collect, store, process, or transmit any personal data or user information.

## What This Plugin Contains

This plugin consists entirely of:

- Markdown instruction files (skills)
- Static documentation reference files
- An agent definition file
- A hook configuration that runs locally

All plugin functionality executes locally within your Claude Code session. No data is sent to external servers, APIs, or third parties.

## Third-Party Services

This plugin does not connect to any third-party services. The Stream Deck plugins you build using this plugin's guidance may connect to external services, but that is governed by your own plugin's privacy practices, not this one.
